# Données France

- **ITV** : Indice Tous Véhicules  
- **IPL** : Indice Poids Lourds  
- **MGL_ITV** : Moyenne Glissante Tous Véhicules  
- **MGL_IPL** : Moyenne Glissante Poids Lourds
